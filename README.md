# 🪐 Python Solar System Simulation

A Python-based Solar System visualization project built using **Tkinter** and **Turtle Graphics**.  
This project simulates planetary motion and simple orbital behavior using Python animation and math logic.

---

## 🚀 Features

- 🌍 Simulates planets orbiting around the sun
- 🎨 Built using Python Turtle graphics
- 🪐 Custom animations with realistic movement logic
- 🎮 Interactive GUI using Tkinter
- 🔄 Uses randomization and math for orbital effects
